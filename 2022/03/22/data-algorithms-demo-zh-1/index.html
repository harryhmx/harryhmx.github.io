100层楼和两个玻璃瓶
Harry Huang
  2022-03-22 03:18:18 2022-03-22 03:18

这是Google的一道面试题，原题是用两个鸡蛋在100层高楼测试，然后用最少的次数确定鸡蛋在哪一层摔碎。因为生活中的实际情况是鸡蛋非常容易碎，通常连第一层楼都抗不住，所以我把题目稍微改动一下，也就是把鸡蛋替换成抗摔的玻璃瓶，详情如下：

  • 有一座100层高的大楼，和两个相同的玻璃瓶，已知玻璃瓶会在某一层丢下摔碎。请你用最少的次数，确定玻璃瓶在哪一层刚好摔碎？

现在开始解这道题，我们来讨论以下几种情况：

  • 二分法

首先，第一个玻璃瓶在第50层测试，如果玻璃瓶没有摔碎，我们就继续在第75层测试，还没摔碎的话，继续在第88层测试，按照50 -> 75 -> 88 -> 94 -> 97 -> 99 -> 100依此类推，假设玻璃瓶在第100层摔碎，我们测试了7次；如果第一个玻璃瓶在第50层摔碎了，这时只剩下一个玻璃瓶，就只能从第1 ～ 49层逐个测试，这样最坏情况是在第49层摔碎，需要测试1 + 49 = 50次；

  • 间隔十层测试

第一个玻璃瓶分别在第10、20、30、40、50、60、70、80、90、100层逐个测试，当第一个玻璃瓶摔碎后，假设它在第40层摔碎，我们就再用第二个玻璃瓶从第31 ～ 39层逐个测试，直到第二个玻璃瓶摔碎为止；这样一来，最坏情况是在第99层摔碎，第一个玻璃瓶从第10 ～ 100层测试了10次，第二个玻璃瓶从第91 ～ 99层测试了9次，总共加起来，需要测试10 + 9 = 19次；但是这个解法依然不是最优解，因为第一个玻璃瓶在不同楼层摔碎的情况下，用第二个玻璃瓶测试再摔碎后，总共测试次数的最大值是不一样的；那么，接下来的第三种解法，我们可以让第一个玻璃瓶在不同楼层摔碎的每一种情况下，第二个玻璃瓶测试摔碎后的总共测试次数最大值相同，即动态规划；

  • 动态规划调整

假设第一个玻璃瓶第1次测试在第m层，如果玻璃瓶摔碎了，我们用第二个玻璃瓶从第1 ～ m - 1层逐个测试，这样最多测试次数为m次；但如果第一个玻璃瓶第1次测试没摔碎，我们为了要让两个玻璃瓶总测试次数控制在m次以内，则第二个玻璃瓶最多测试次数为m - 1次，这样第一个玻璃瓶第2次测试需要放在第m + (m - 1) = 2m - 1层；而如果第一个玻璃瓶第2次测试还没摔碎，就需要再往上进行第3次测试，第3次测试需要放在第m + (m - 1) + (m - 2) = 3m - 3层；根据规律以此类推，假设第一个玻璃瓶用了m - 1次测试仍未摔碎，这时我们就只能再进行最后一次测试了，这样最后一次测试所在的层数N为
