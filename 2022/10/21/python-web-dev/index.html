Python Web Development
Harry Huang
  2022-10-21 13:10:06 2022-10-21 13:10

  • Python3.7 Installations
    $ sudo apt-get install software-properties-common
    $ sudo apt-get install python3-distutils
    $ sudo add-apt-repository ppa:deadsnakes/ppa
    $ sudo apt-get update
    $ sudo apt-get install python3.7

  • PIP & Virtualenv Installations
    $ mkdir wgets && cd wgets
    $ wget https://bootstrap.pypa.io/get-pip.py
    $ sudo python3 get-pip.py
    $ pip3 --help
    $ sudo pip3 install virtualenv

  • Using Virtualenv & Install Django
    $ cd .. && mkdir virtual && cd virtual
    $ virtualenv roamerharry
    # Create virtual python environment named ‘roamerharry’
    OR     $ virtualenv --python=python3.7 roamerharry
    # Create virtual python environment named ‘roamerharry’ for version 3.7
    $ source roamerharry/bin/activate
    # Enter environment ‘roamerharry’
    $ pip install django
    OR $ pip install django==2.2.15
    # Install Django
    $ deactivate
    # Quit environment ‘roamerharry’

  • Start project
    $ cd .. && mkdir projects && cd projects
    $ django-admin.py startproject roamerharry
    # roamerharry: project_name
    $ cd roamerharry

  • Run project
    $ python manage.py runserver
    or
    $ python manage.py runserver [ip]:[port]
    example
    $ python manage.py runserver 0.0.0.0:8024

  • Create new app
    $ python manage.py startapp archive
